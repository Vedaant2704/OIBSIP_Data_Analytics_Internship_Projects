# Wine Quality Prediction using Machine Learning

## Project Overview
This project predicts wine quality based on chemical properties using multiple classification models.

---

## Models Used
- Random Forest
- Support Vector Machine
- SGD Classifier

---

## Key Findings
- Alcohol content is the strongest quality predictor
- Random Forest achieved the best performance

---

## Conclusion
The project demonstrates a complete machine learning pipeline with model comparison and interpretation.
