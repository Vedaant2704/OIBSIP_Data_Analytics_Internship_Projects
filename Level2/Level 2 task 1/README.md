# House Price Prediction using Linear Regression

## Project Overview
This project focuses on predicting house prices using Linear Regression by analyzing how various housing features influence property prices. It provides hands-on experience with regression modeling and evaluation techniques.

---

## Dataset Description
The dataset includes features such as house area, number of rooms, parking, furnishing status, and price.

---

## Workflow
- Data loading and exploration
- Data preprocessing
- Exploratory data analysis
- Model training using Linear Regression
- Model evaluation using MAE, MSE, RMSE, and RÂ²

---

## Key Results
- Area and furnishing status significantly impact price
- Linear Regression serves as a strong baseline model

This project strengthened foundational regression and predictive modeling skills.
