# Google Play Store App Analysis

## Project Overview
This project analyzes Google Play Store data to understand app performance, ratings, pricing, and user behavior.

---

## Key Insights
- Most apps follow a freemium model
- App performance matters more than price
- Niche categories offer strong opportunities

---

## Conclusion
This project highlights storytelling with data and business-oriented analytics.
