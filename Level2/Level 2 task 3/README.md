# Fraud Detection Using Data Analytics & Machine Learning

## Project Overview
This project focuses on identifying fraudulent transactions using machine learning, with special emphasis on recall due to business impact.

---

## Models Used
- Logistic Regression
- Random Forest

---

## Key Insights
- Fraud datasets are highly imbalanced
- Random Forest significantly improves fraud detection
- Recall is the most critical metric

---

## Conclusion
The project aligns technical modeling with real-world business priorities.
